1) Create a virtual env(for linux):
   code:

   virtualenv .env

   
2) Activte virtual env(linux):
   code:

   source .env/bin/activate
3) Change dir to main:
   code:

   cd main

   
4) Install rqmts:
   code:

   pip install -r requirements.txt


5) Start server:
   code:

   python manage.py runserver
